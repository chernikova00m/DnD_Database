Главная

Добро пожаловать на сайт проекта DnD Character Database. Здесь представлена база данных для управления персонажами настольной ролевой игры Dungeons & Dragons, разработанная в рамках проектной практики Московского Политеха.

О проекте

Цель: создание удобного инструмента для хранения и управления характеристиками игровых персонажей.

Технологии: C# (.NET 6+), SQLite, Git, Markdown, HTML/CSS.

Приложение позволяет добавлять, просматривать, фильтровать по классам и удалять персонажей, автоматически сохраняя данные в локальную базу данных.
